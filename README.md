### Overview
Synestia is a financial planning and analysis (FP&A) tool designed to help SMBs automate the visualization of P&L data and model financial scenarios in real-time. Built to replace static Excel spreadsheets with dynamic, interactive insights.

**Project Status:** MVP (Minimum Viable Product)

### ðŸ“Š Key Features
1.  **Instant P&L Visualization:** Upload a raw CSV and instantly generate revenue vs. expense trend lines and profit margin analysis.
2.  **Interactive Scenario Planner:** A dynamic slider allows users to simulate "What-If" scenarios (e.g., *How does a 5% reduction in OpEx impact Net Income?*).
3.  **Automated Variance Analysis:** (In Progress) Logic built to identify risk factors, such as expense growth outpacing revenue.
